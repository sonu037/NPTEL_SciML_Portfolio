# Scientific Machine Learning (SciML)
**Course:** NPTEL - Machine Learning for Engineering and Science Applications (IIT Madras)
**Status:** In Progress (Jan 2026)

## 📌 Project Objective
Implementing Machine Learning algorithms specifically for **Physics-Based Problems** (ODEs, PDEs, and Regression), moving beyond standard "Data Science" into "Engineering AI."

## 🛠️ Tech Stack
* **Frameworks:** TensorFlow / PyTorch, Scikit-Learn.
* **Math:** NumPy, Pandas.

## 📂 Key Implementations
1.  **Physics-Informed Regression:** Fitting curves to noisy sensor data while respecting physical constraints.
2.  **ODE Solver with Neural Networks:** Using a Neural Network to solve differential equations (d/dx = -ky).
3.  **Surrogate Modeling:** Replacing expensive simulations with fast ML approximators.
